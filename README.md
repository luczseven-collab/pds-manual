# PDS アンケート自動集計・AI回答システム

PARTY DRESS STYLEの紙アンケートをスキャンしてGoogleドライブに保存するだけで、OCR読み取り・選択肢集計・手書き感想の文字起こし・AI自動回答・意見分析までを自動化するシステム。

## 概要

- **スキャン → Drive保存** だけで全処理が自動実行される
- Cloud Vision APIで選択肢（○印）を座標判定
- Claude APIで手書きテキスト補正・感情分析・AI回答生成
- Google Sheetsに集計・意見・ログを自動書き込み

## システムフロー

```
スキャン画像 → Google Drive → GASトリガー検知 → Cloud Vision OCR
→ 選択肢判定 → 集計シート書き込み
→ 手書きテキスト補正（Claude API）
→ 感情分析 → ネガティブハイライト
→ AI回答生成（承認フロー）
→ 意見まとめ・分析シート更新
→ Chatwork通知
```

## 開発フェーズ

| フェーズ | 内容 | 期間 |
|---------|------|------|
| Phase 1 | Driveフォルダ監視 + OCR + 選択肢判定 + 集計シート | 3〜4週間 |
| Phase 2 | 手書き補正 + 感情分析 + AI回答 + 意見まとめ | 2〜3週間 |
| Phase 3 | 意見分析 + トンマナ設定 + 通知 + ログ + マニュアル | 1〜2週間 |

## セットアップ

1. Google Apps Scriptプロジェクトを作成
2. `src/gas/` 配下のファイルをコピー
3. スクリプトプロパティにAPIキーを設定
4. Cloud Vision APIを有効化
5. Google Driveの監視フォルダIDをConfigに設定
6. タイマートリガー（5〜10分間隔）を設定
