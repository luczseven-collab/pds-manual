# Claude API 月額費用試算

## 前提条件

| 項目 | 内容 |
|------|------|
| 使用モデル | claude-opus-4-6 |
| Input単価 | $15 / 1Mトークン |
| Output単価 | $75 / 1Mトークン |
| 為替レート | 1ドル = 150円 |

## 1枚あたりのコスト実測値

| 処理パターン | Inputトークン | Outputトークン | 費用（ドル） | 費用（円） |
|------------|-------------|--------------|------------|----------|
| 自由意見あり（画像読取＋感情分析＋返信文生成） | 約3,600 | 約470 | $0.09 | 約13円 |
| 自由意見なし（画像読取のみ） | 約2,960 | 約210 | $0.06 | 約9円 |

## まとめ

- **想定運用（月300枚・自由意見50%）：約3,400円/月**
- 自由意見なしのアンケートはコストが約30%削減
- 繁忙期（月500枚）でも最大約6,750円/月に収まる見込み
